## 合計
- **必須**: 10ファイル
- **オプション**: 1ファイル（フォールバック）

## 画像形式
- **推奨形式**: JPG（ファイルサイズを抑えるため）
- **代替形式**: PNG（透明度が必要な場合）
- **最適化**: Web用に圧縮（200KB以下を推奨）

## 注意事項
- ファイル名は大文字小文字を区別します
- パスは `/images/ファイル名.jpg` の形式で参照されます
- Viteを使用しているため、`public`フォルダ内のファイルはルートパス（`/`）から直接アクセス可能です
